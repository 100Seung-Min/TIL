# 파이썬 기본 문법
---
1. 입출력
> 출력은 print   
> 입력은 input   
> 입력 구분은 .split()
```py
print('안녕')
a = input().split()
```
2. 수식
> \+ -> 더하기   
> \- -> 빼기   
> / -> 나누기   
> \* -> 곱하기   
> % -> 나머지   
> // -> 버림   
> ** -> 거듭제곱      
> 자료형(값) -> 값을 자료형으로 만듦   
> type(값) -> 객체의 자료형을 알려줌   
> (값) -> 값을 우선연산   
```
a = 2 ** 4
print(type a)
print(int(5.8+3.1))
```
3. 변수
> 변수의 이름 영문과 숫자, _   
> 파이썬의 키워드는 사용불가
> type(값) -> 변수의 자료형을 알려줌
