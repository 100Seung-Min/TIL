### 최솟값 만들기
```py
def solution(A,B):
    c = A
    d = B
    A.sort()
    B.sort(reverse=True)
    c.sort(reverse=True)
    d.sort()
    answer1 = 0
    answer2 = 0
    for i, v in enumerate(A):
        answer1 = answer1 + (v * B[i])
        answer2 = answer2 + (c[i] * d[i])
    return answer1 if answer1 < answer2 else answer2
```
