# 클로저
---
### 1. String을 반환하는 클로저
```swift
let myName: String = {
  return "정대리"
}()

print(myName)
```

### 2. 매개변수를 반환하는 클로저
```swift
let myRealName = { (name: String) -> String in
  return "개발하는 \(name)"
}()

myRealName
```

### 3. 클로저 로직
```swift
let myRealNameLogic = { (name: String) -> Void in
  return print("개발하는 \(name)")
}()

myRealNameLogic
