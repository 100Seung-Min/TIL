# 인공지능 용어
## 1. 선형회귀
> 입력 데이터와 타깃 데이터를 이용해 기울기와 절편을 찾는것   
> 식으로는 y = ax + b   
> 구하는 법은 경사 하강법, 정규 방정식, 결정 트리, 서포트 벡터 머신 등 많이 있다.
## 2. 알고리즘식
> y = ax + b에서 a -> 가중치나 계수로 표현, y는 와이-햇으로 표현한다.   
> 절편 b와 가중치 w는 규칙을 의미하고, 와이-햇은 예측한 값을 의미한다.
## 3. 에포크
> 경사하강법에서 한가지의 데이터로 여러번의 학습을 하는 것
## 4. 오즈 비
> 성공확률과 실패확률
